An interactive Weather Dashboard built with **Python** and **Streamlit** that allows users to explore real-time weather conditions and forecasts through a clean and intuitive interface. The app fetches live weather data from an API and visualizes key information such as temperature, conditions, and trends using dynamic charts and tables.

This project demonstrates **API integration, data processing, and interactive dashboard design**, making it useful for users who want quick insights into weather conditions worldwide.

---

## 🚀 Features
- 🌍 Search weather by city or location  
- 📊 Interactive charts and visualizations  
- 🌡️ Real-time temperature and weather conditions  
- 📅 Forecast display for upcoming hours or days  
- ⚡ Built with Python, Streamlit, and APIs  

---

## 🛠️ Tech Stack
- Python 3  
- Streamlit  
- Pandas  
- Weather API  
- Plotly / Matplotlib
